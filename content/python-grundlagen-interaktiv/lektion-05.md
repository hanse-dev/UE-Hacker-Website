# Bedingungen mit if

Mit `if` führt Python Code nur aus, wenn eine Bedingung wahr ist.

```python
if alter >= 16:
    print("Du bist mindestens 16!")
```

Die Einrückung ist wichtig – sie gehört zum `if`-Block.

**Aufgabe 1:** Gib „Du bist mindestens 16!“ aus, wenn eine Variable `alter` mindestens 16 ist.

**Aufgabe 2:** Nutze `if` und `else`: Wenn `alter` unter 16 ist, gib „zu jung“ aus; sonst „Du bist mindestens 16!“.
