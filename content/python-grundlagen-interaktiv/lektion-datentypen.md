# Datentypen

Python kennt verschiedene Datentypen: Ganzzahlen (`int`), Dezimalzahlen (`float`), Texte (`str`) und Wahrheitswerte (`bool`). Mit `type()` kannst du den Typ eines Werts herausfinden.

```python
print(type(42))      # <class 'int'>
print(type(3.14))   # <class 'float'>
print(type("Hallo")) # <class 'str'>
print(type(True))   # <class 'bool'>
```

**Aufgabe 1:** Gib den Typ der Zahl 100 aus (nutze `type()`).

**Aufgabe 2:** Gib den Typ des Textes „Python“ aus.
