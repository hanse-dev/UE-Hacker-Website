# Dictionaries

Ein Dictionary speichert Schlüssel-Wert-Paare. Du greifst mit dem Schlüssel auf den Wert zu.

```python
woerterbuch = {"apfel": "apple", "buch": "book"}
print(woerterbuch["apfel"])
```

**Aufgabe 1:** Erstelle ein Dictionary mit „apfel“ → „apple“ und „buch“ → „book“. Gib den Wert zu „apfel“ aus.

**Aufgabe 2:** Gib den Wert zu „buch“ aus (aus demselben Dictionary).
