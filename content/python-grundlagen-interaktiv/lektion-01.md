# Dein erstes Programm

Mit `print()` gibst du Text auf dem Bildschirm aus. Der Text muss in Anführungszeichen stehen.

Mit `#` kannst du Kommentare schreiben – Python ignoriert alles danach bis zum Zeilenende. Kommentare helfen dir, den Code zu verstehen.

**Aufgabe 1:** Gib „Hallo Welt“ aus.

**Aufgabe 2:** Gib „Fertig“ aus.
