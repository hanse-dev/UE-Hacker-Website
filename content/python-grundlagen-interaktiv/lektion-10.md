# Kleine Übung: Alles zusammen

Kombiniere, was du gelernt hast: Variablen, Schleifen, Listen und Funktionen.

```python
zahlen = [1, 2, 3]
for z in zahlen:
    print(z * 2)
```

**Aufgabe 1:** Erstelle eine Liste mit Zahlen und gib jede Zahl verdoppelt aus (z.B. [1,2,3] → 2, 4, 6).

**Aufgabe 2:** Erstelle eine Funktion `summe(zahlen)`, die eine Liste von Zahlen nimmt, die Summe berechnet und mit `print()` ausgibt. Rufe sie mit `[1, 2, 3, 4, 5]` auf. Die Ausgabe soll 15 enthalten.
