# Variablen

Variablen sind Speicher für Werte. Du weist einen Wert zu und kannst ihn später verwenden.

```python
name = "Python"
print(name)  # gibt "Python" aus
```

**Aufgabe 1:** Speichere „Python“ in einer Variable und gib sie mit `print()` aus.

**Aufgabe 2:** Speichere die Zahl 42 in einer Variable und gib sie aus.
