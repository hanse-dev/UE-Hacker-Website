# Listen

Listen speichern mehrere Werte. Der erste Eintrag hat den Index 0, der letzte z.B. Index -1.

```python
tiere = ["Hund", "Katze", "Maus"]
print(tiere[0])   # gibt "Hund" aus
print(tiere[-1])  # gibt "Maus" aus (letztes Element)
```

**Aufgabe 1:** Erstelle eine Liste mit „Hund“, „Katze“, „Maus“ und gib das erste Element aus.

**Aufgabe 2:** Gib das letzte Element derselben Liste aus (Index -1).
