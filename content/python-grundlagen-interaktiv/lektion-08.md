# Funktionen

Funktionen sind wiederverwendbare Code-Blöcke. Du definierst sie mit `def` und rufst sie mit `()` auf.

```python
def gruessen():
    print("Guten Tag!")

gruessen()
```

**Aufgabe 1:** Definiere eine Funktion, die „Guten Tag!“ ausgibt, und rufe sie auf.

**Aufgabe 2:** Definiere eine Funktion `verdoppeln(zahl)`, die eine Zahl verdoppelt und ausgibt. Rufe sie z.B. mit 3 auf (Ausgabe: 6) oder mit 3.5 (Ausgabe: 7.0).
