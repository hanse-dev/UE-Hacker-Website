# Zahlen und Rechnen

Python kann rechnen: `+`, `-`, `*`, `/` funktionieren wie gewohnt.

```python
a = 10
b = 3
print(a + b)
```

**Aufgabe 1:** Erstelle zwei Variablen mit Zahlen und gib ihre Summe aus (z.B. 10 + 3 = 13).

**Aufgabe 2:** Erstelle zwei Variablen und gib ihr Produkt aus (z.B. 10 * 3 = 30).
