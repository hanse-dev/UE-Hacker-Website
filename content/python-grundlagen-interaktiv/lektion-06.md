# Schleifen mit for

Mit `for` wiederholst du Code. `range(3)` liefert die Zahlen 0, 1, 2.

```python
for i in range(3):
    print("Hallo")
```

**Aufgabe 1:** Gib „Hallo“ mehrfach aus (z.B. mit einer for-Schleife über `range`).

**Aufgabe 2:** Nutze eine for-Schleife und gib die Schleifenvariable aus (z.B. `for i in range(3): print(i)` – die Ausgabe enthält 0).
