# Strings formatieren mit f-Strings

Mit f-Strings kannst du Variablen direkt in Text einbinden. Schreibe `f` vor die Anführungszeichen und `{variable}` für den Wert.

```python
name = "Python"
print(f"Hallo, {name}!")
```

**Aufgabe 1:** Erstelle eine Variable und gib sie mit einem f-String aus (z.B. „Hallo, [Name]!“).

**Aufgabe 2:** Erstelle eine Variable `alter` mit einer Zahl und gib z.B. „Ich bin X Jahre alt“ aus (mit f-String).
